# web_app
User sign in and sign up

# Command
There are two folders: frontend and backend.  
Make sure you have installed 'node.js' and 'npm'.  

1. In frontend folder:  
$ cd web_app  
$ cd frontend  
$ cnpm i  
$ npm start  

2. In backend folder:  
$ cd backend  
$ cd npm install   
$ node app  

Wait a few seconds for web loading
